## 🚀 方式一：本地查看（最简单）

直接双击 `D:\作品集\index.html`，浏览器打开即可浏览完整作品集。

**优点**：无需上传、完整剪纸风格、视频即点即看
**适用**：本地预览、投屏展示

---

## 🌐 方式二：部署在线 + 飞书链接分享

### 步骤 1：部署网页
将整个 `D:\作品集` 文件夹上传到任意静态托管：

| 平台 | 说明 |
|------|------|
| **GitHub Pages** | 免费，创建仓库后上传文件夹，开启 Pages |
| **Vercel** | 免费，拖拽部署 |
| **Netlify** | 免费，拖拽部署 |
| **腾讯云COS** | 国内速度快，需备案 |

### 步骤 2：获取链接
部署后会得到一个网址（如 `https://你的域名/作品集/`）

### 步骤 3：飞书分享
在飞书中直接粘贴链接，飞书会自动生成预览卡片。面试官点击即可在浏览器中打开。

**优点**：完整设计效果、视频流畅播放、可分享给任何人
**适用**：远程面试、飞书投递

---

## 📄 方式三：飞书文档版（手动迁移）

如果你想在飞书原生文档中创建作品集，步骤如下：

1. **新建飞书文档** → 标题设为"江楷煜 · AIGC设计师作品集"
2. **上传视频到飞书云盘** → 在文档中插入「视频」模块
3. **上传海报图片** → 在文档中插入「图片」模块
4. **按作品集结构排版**：精选视频 → AI设计 → 区块链运营 → 项目 → 实习经历
5. **生成分享链接** → 设置权限为「获得链接的人可查看」

**注意**：飞书文档不支持自定义CSS，无法实现剪纸风格效果。建议优先使用方式一或方式二。

## ❓ 常见问题

**Q: 视频无法播放？**
A: 确保视频文件和 index.html 在同一目录结构中。路径不要改动。

**Q: 海报图片不显示？**
A: 检查「海报」文件夹是否完整，路径是否与 index.html 中一致。

**Q: 飞书文档嵌入视频有大小限制吗？**
A: 飞书免费版单个视频不超过 2GB，你的视频都在 100MB 以内，没问题。

**Q: 可以只在飞书中分享链接吗？**
A: 可以！把网页部署后，把链接复制到飞书消息中即可。
